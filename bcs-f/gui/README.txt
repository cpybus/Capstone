Base directory for the gui Django project.

Overview
	This directory contains a Django project which subscribes to live data coming from the servers and generates an RTAP display of the information. It provides features to the operator and displays the live airspace.

Directory Listing
	- gui: Contains the Django project files
	- rtap: Contains the static files which generate the actual UI display

Listing of Important Files
	- manage.py: This is a Django generated file which is used for running the server
